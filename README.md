
## 📂 Lab Projects

### 1. Buck Converter
- **Objective:** Study step-down DC-DC conversion and analyze voltage regulation and inductor current.  
- **Report:** [Buck Converter Report](./Buck_Converter/Report.md)  
- **Figures:** Circuit diagram, output voltage, inductor current.  
- **Simulations:** MATLAB Simulink files included.

### 2. Flyback Converter
- **Objective:** Understand isolated DC-DC conversion using a Flyback topology.  
- **Report:** [Flyback Converter Report](./Flyback_Converter/Report.md)  
- **Figures:** Transformer diagram, output voltage, inductor current.  
- **Simulations:** MATLAB Simulink files included.

### 3. Closed Loop PWM Controller
- **Objective:** Implement a closed-loop PWM controller to maintain constant output voltage under varying loads.  
- **Report:** [Closed Loop PWM Controller Report](./Closed_Loop_PWM_Controller/Report.md)  
- **Figures:** Circuit diagram, output voltage, inductor current.  
- **Simulations:** MATLAB Simulink files included.
